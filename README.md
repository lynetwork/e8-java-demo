# e8-java-demo

#### 介绍
易呗云收银JAVA语言开发DEMO

#### 软件架构
	- config 配置目录
		- Config.java 配置文件
		
	- controllers 控制器目录（接口）
		- PayController.java 调试接口与回调
		
	- entity 实体类目录
		- Callback.java 回调参数实体
		- Order.java 下单参数实体
		
	- helper 工具类目录
		- Helper.java 工具类
		- MD5.java MD5工具类
		- RSAUtil.java RSA签名工具类
		- SignUtil.java 签名工具类
		- SignUtils.java 签名工具类
		- XmlUtils.java xml工具类

1.  eclipse
2.  maven
